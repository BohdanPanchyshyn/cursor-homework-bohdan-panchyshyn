    • git config --global user.email blabal@bla - Добавити свій email адрес. git init - Ініціалізація репозиторію з локальною папкою. git status - перевірка стану файлів. git add - додавання змін у файли. git commit - збереження змін у файлах. git log - перегляд історії змін файлів. git checkout - перехід між гілками/репозиторіями. git branch - створення нової гілки. git push - залити данні на GITHUB. git pull - стянути данні з GITHUB. git merge - Об'єднати данні гілок. git clone - клонування репозиторію собі на комп'ютер. gitignore - папка синхронізації(або відстежування папки на локальній мамшині), без неї неможливо синхронізуватись між локальним ПК і GITHUB. git revert - відміна зміни і комітів. git reset - git rebase - git cherry-pick - git stash -
